# Quick Demo Instructions

## How to Run the E-Commerce Demo

1. **Open the Application**
   - Simply open `index.html` in any modern web browser
   - No server setup required - it runs entirely in the browser

2. **Test the Shopping Experience**
   - Browse the 6 sample products
   - Click "Add to Cart" on any product
   - Click the cart icon (top right) to view your cart
   - Adjust quantities using +/- buttons
   - Remove items using the trash icon

3. **Complete a Purchase**
   - Click "Proceed to Checkout" in the cart
   - Fill out the customer information form
   - Select a payment method
   - Click "Complete Payment"
   - Wait 2 seconds for payment processing simulation

4. **View and Manage Invoice**
   - After payment, an invoice will automatically appear
   - Click "Download PDF" to save as text file
   - Click "Print" to open print dialog
   - Click the X to close the invoice

## Technical Notes

- All data is stored in browser memory (lost on refresh)
- Payment processing is simulated (2-second delay)
- Invoice download creates a text file (PDF requires server-side processing)
- Images are loaded from Unsplash (requires internet connection)
