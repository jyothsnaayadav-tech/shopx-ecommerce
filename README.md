# ShopX E-Commerce App

A premium, full-stack mimicking e-commerce web application built entirely within a single-page architecture. 

## 🌟 Features
- **Authentication**: Seamless Login / Registration flows.
- **Dynamic Shopping Cart**: Slide-out cart sidebar with live quantity updates and total calculations.
- **Wishlist Engine**: Heart products to save them for later.
- **Simulated Checkout & Invoices**: Generates highly styled, printable/PDF exportable transaction invoices simulating real checkout sequences.
- **Admin Dashboard**: Full CRUD (Create, Read, Update, Delete) capability for products, order status tracking, and user privilege management.
- **Review System**: Five-star rating system with verified customer reviews.

## 🛠️ Tech Stack
This project does not rely on traditional API backends! It focuses on pure frontend performance and caching techniques.
- **HTML5** (Structuring the SPA layout)
- **CSS3 / Tailwind CSS** (Leveraging CDN for premium dark-mode styling and glassmorphism)
- **Vanilla JavaScript** (DOM manipulation, Routing mechanisms, and real-time computation)
- **LocalStorage DB** (Acting as a synchronous, localized SQLite simulation replacing the need for node/express)

## 🚀 How to Run locally
1. You do **not** need Node.js, Python, or a live server environment.
2. Simply download or clone this repository.
3. Open `index.html` in any modern web browser (Google Chrome, Firefox, Safari).
4. The database instantly seeds itself!
